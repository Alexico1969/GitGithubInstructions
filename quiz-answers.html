Quiz Answers

Here are the answers to the quiz questions:

1. What command do you use to initialize a Git repository?

Answer: git init

2. How do you create a new private repository on GitHub?

Answer: Go to your GitHub account, click the "+" icon in the top-right corner, select "New repository," enter a name, and choose "Private."

3. What command do you use to push your local repository to GitHub?

Answer: git push origin main

4. What is a personal access token used for?

Answer: A personal access token is used for authentication when pushing code to GitHub, replacing your password.

5. Why is it important to keep your repository private?

Answer: Keeping your repository private ensures that sensitive or personal information is not publicly accessible.

6. What command do you use to check the status of your Git repository?

Answer: git status

7. How do you add files to your Git staging area?

Answer: git add . (to add all files) or git add [filename] (to add a specific file).

8. What command do you use to commit changes to your local repository?

Answer: git commit -m "Your commit message"

9. How do you check the commit history of your Git repository?

Answer: git log

10. What is the purpose of the README.md file in your repository?

Answer: The README.md file provides an overview of the project, including its purpose, usage instructions, and other relevant details.
